HTML-学习笔记

2020.06.28

什么是HTML？

HTML是「HyperText Mark-up Language」的首字母缩写，中文全称是「超文本标记语言」，是一种用于创建网页的标记语言。它不是一种编程语言，而是一种标记语言「markup language」。

什么是HTML标签？

标记语言是一套标记标签「markup tag」，HTML标记标签通常被称为 HTML 标签。

HTML 标签有尖括号包裹关键词，如：<html> 、<head> 、<body> 等等。

HTML 标签通常是成对出现的，比如：

<p>...</p>

标签对中的第一个标签是开始标签，第二个标签是结束标签，结束标签具有 / 。

此外，HTML 标签也有单独出现的标签，比如：

<meta>
<br>
<img>

HTML 文档结构

<!DOCTYPE html><!--!:表示申明的意思。这一行代码的意思是：下面的文档标签将以html5规范去解析-->
<html>
<head><!--1.头部，主要用来完成一个网页的相关设定的。-->
    <meta charset="utf-8"><!--编码方式--> <!--meta:元，主要用来完成对应设定的-->
    <meta name="keywords" content=""><!--设定一个网站的搜索关键字-->
    <meta name="description" content=""><!--网站的描述内容-->
    <title>我是一个网页的标题</title><!--标题-->
    <!--下面link引入的是设定网站小图标-->
    <link rel="shortcut icon" href="https://img.alicdn.com/tps/i3/T1OjaVFl4dXXa.JOZB-114-114.png" type="image/png">
    <style>
        /*书写样式的地方*/
    </style>
    <link rel="stylesheet" href="style.css"><!--用来引入外部样式文件-->
</head>
<body><!--2.主体部分-->
    <p>这是一个段落</p>
</body>
<script>
    // 放脚本代码的地方
</script>
</html>

  • <html></html>称为根元素，所有的网页元素都在<html> </html> 中；

  • <head></head>元素用于定义文档的头部；

    头部元素含有：<base> <meta> <title> <link> <script> <style> ；

    • <title>...</title> 用于定义网页的标题，标题写在 <title> 和 </title> 之间；
    • <meta> 提供关于 HTML 文档的元数据，元数据不会显示在页面上，但对浏览器是可读的；
    • <link> 用于来引入页面图标、外部CSS文件等；
    • <script></script> 用于放置或引入 JavaScript 脚本内容；
    • <style></style> 用于为 HTML 文档定义样式信息；
    • <base> 用于为页面上的所有链接规定默认地址或默认目标「target」。

  • <body></body> 元素用于定义页面显示的内容，将需要呈现的内容编辑于 <body> 和 </body> 之间；

HTML 中 head 元素内的常用设定

meta 元素
  • 编码格式
<meta charset="UTF-8">

HTML 5 设定网页编码格式的方式。

来自维基百科：UTF-8「8-bit Unicode Transformation Format」是一种针对Unicode的可变长度字符编码，也是一种前缀码。自2009年以来，UTF-8一直是万维网的最主要的编码形式，截止到2019年11月， 在所有网页中，UTF-8编码应用率高达94.3%。第二热门的多字节编码方式 Shift JIS 和 GB 2312 分别具有0.3％和0.2％的占有率。

<meta http-equiv="content-Type" content="text/html; charset=utf-8">

旧版本 HTML 设定编码格式的方式，现已不推荐，更多的是使用 HTML 5 的设定方式。

  • keywords (关键字)
<meta name="keywords" content="博客">

用于告诉搜索引擎，您网页的关键字。

  • description (网站内容的描述)
<meta name="description" content="">

用于告诉搜索引擎，您网页的主要内容。

  • viewport (移动端的窗口)
<meta name="viewport" content="width=device-width, initial-scale=1">

用于设计移动端网页时做的设定。

“width=device-width”：宽度=设备宽度

“initial-scale=1.0”：初始比例=100%

width：控制页面在设备荧幕上可视区域的宽度，可以指定一个值。

height：同上，控制页面在设备荧幕上可视区域的高度。

initial-scale：初始缩放比例。

maximum-scale：允许用户缩放到的最大比例。

minimum-scale：允许用户缩放到的最小比例。

user-scalable：用户是否可以手动缩放。

常见的设定大概是：

<!-- 指定荧幕宽度为设备宽度，画面载入初始缩放比例 100% -->
<meta name="viewport" content="width=device-width, initial-scale=1">

<!-- 以下两种方法都可以防止用户做画面缩放，将画面缩放比例锁定在 100% -->
<meta name="viewport" content="width=device-width, initial-scale=1, minimum-scale=1, maximum-scale=1">

<meta name="viewport" content="width=device-width, initial-scale=1, user-scalable=no">

参考资料：Responsive Web Design 基礎 : 設定

  • author (作者)
<meta name="author" content="张三">

用于标注网页的作者。

  • robots (定义搜索引擎爬虫的索引方式)
<meta name="robots" content="none">

用于告诉爬虫哪些页面需要索引，哪些页面不需要索引。

none：搜索引擎将忽略此网页，等价于noindex，nofollow。

noindex：搜索引擎不索引此网页。

nofollow：搜索引擎不继续通过此网页的链接索引搜索其他网页。

all：搜索引擎将索引此网页并且继续通过此网页的链接索引其他网页，等价于index，follow。

index：搜索引擎索引此网页。

follow：搜索引擎继续通过此网页的链接索引搜索其他网页。

  • revisit-after (搜索引擎爬虫重访时间)
<meta name="revisit-after" content="7 days">

用于设定搜索引擎爬虫重访时间，如果页面不是经常更新，为了减轻搜索引擎爬虫对服务器带来的压力，可以设定该属性。如果设定的重访时间过短，爬虫将按它们定义的默认时间来访问。

  • copyright (版权)
<meta name="copyright" content="">

用于标注版权信息，代表该网站为 XX 个人版权所有。

  • X-UA-Compatible (浏览器采取何种版本渲染当前页面)
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">

用于告知浏览器以何种版本来渲染页面。

IE=edge, chrome=1：指定 IE 和 Chrome 都使用最新版本来渲染当前页面。

  • expires (网页到期时间)
<meta http-equiv="expires" content="Sunday 26 October 2016 01:00 GMT">

用于设定网页的到期时间， 过期后网页必须到服务器上重新传输。

  • Set-Cookie (cookie设定)
<!-- 格式 -->
<meta http-equiv="Set-Cookie" content="name, date">

<!-- 例子 -->
<meta http-equiv="Set-Cookie" content="User=xxxxx; path=/; expires=Sunday, 10-Jan-16 10:00:00 GMT">

如果网页过期。那么这个网页存在本地的cookies也会被自动删除。

  • refresh (自动刷新并指向某页面)
<meta http-equiv="refresh" content="2；URL=http://www.xxxxx.com/">

用于设定网页自动刷新且跳转到指定网站。

content="2；URL=http://www.xxxxx.com/” ：意思是 2 秒后将刷新并且跳转到指定的网址。

title 元素
  • 网页标题
<title>我是这个网页的标题</title>

定义网页的标题。

link 元素
  • 引入外部样式文件
<link rel="stylesheet" type="text/css" href="style.css">

用于引入外部CSS文件。

  • 加载图标
<link rel="shortcut icon " type="images/x-icon" href="favicon.ico">

用于加载网页的图标。

style 元素
  • 设定元素的样式
<style>
    div {
        width: 100px;
        height: 100px;
    }
</style>

用于为元素定义样式。

script 元素
  • 设定 JavaScript 脚本内容
<script>
	function f() {
    	document.write('script 标签中可以书写JS代码');
    }
    f();
</script>

用于定义页面的脚本内容。

  • 引入外部 JavaScript 文件
<script type="text/javascript" src="ss.js"></script>

用于引入外部 JavaScript 文件。注意：一旦该组 script 标签用于引入外部 JS 文件后，再在其标签中书写 JS 代码将不被浏览器识别。向重新创建新的 script 标签中添加 JS 代码则不会受到影响。

块级元素和内联元素

大多数 HTML 元素被定义为块级元素或内联元素。

块级元素占据其父元素（容器）的整个空间，因此创建了一个“块”。通常浏览器会在块级元素前后另起一行。能容纳其他块元素或者内联元素。块级元素的宽度默认为整行宽度，高度为内容实际的高度。

内联元素在显示时通常不会以新行开始。并且它的宽度和高度分别为内容实际的宽度和高度

区别：

  • 块级元素，总是在新行上开始；内联元素，和其他元素在一行；

  • 块级元素，能容纳其他块级元素或者内联元素；内联元素，只能容纳文本或其他内联元素；

  • 块级元素中高度，宽度都可以控制；内联元素中高度，宽度不可改变。

通过 display 可以进行转换。

举个栗子：

<style>
	div {
		background-color: #ff6239;
	}
	span {
		background-color: #46a6e8;
	}
</style>

<div>div is block level element</div>
<span>span is inline element</span>

HTML 标签属性

通用属性
  • id
<div id="div1"></div>
<div id="div2"></div>

规定元素的唯一 id。id 属性是唯一的，设定 id 时，一个元素对应一个唯一的 id ，别的元素将无法再使用该 id。

  • class
<h2 class="class1">这里是标题</h2>
<p class="class1">这里是内容</p>

规定元素的类名。类名是可重复的，可将需要设定相同样式的元素应用同一类名。

  • style
<div style="width: 200px; height: 200px; background-color:#ff9632;"></div>

规定元素的行内样式。

  • title
<div style="width: 200px; height: 200px; background-color:#ff9632;" title="鼠标放上来将显示此段信息"></div>

规定元素的额外信息，当鼠标放在元素上，提示的文本内容。

自定义属性
  • data-*
<!--以下代码是实际开发的案例，img标签显示的是缩略图，而data-src的值为点击显示原图的地址-->
<img src="image/01.jpg" data-src="image/big_01.jpg">

HTML5规范中指定如果需要使用自定义属性，需要用 data-* 的方式，比如：data-src data-index

data-src 属性是HTML5中引入的 data- * attributes 集合的一部分。 data-src 允许我们存储对浏览器没有意义但可以由 JavaScript 代码或 CSS 规则使用的额外数据。

标签的私有属性
  • HTML 标签中，不同的标签具有不同的私有属性，比如：
<a href="http://www.google.com">这是一个链接</a>

<a> 标签的 href 属性。

<img src="small_01.jpg" alt="">

<img> 标签的 src 属性。

<input type="text" name="" placeholder="请输入您的账号">
<input type="submit" value="登录">

<input> 标签的 type 、name 、value 、placeholder 等等。

HTML 常用的实体转义字符

在HTML中，内容编辑时，如果是通过空格键编辑的多个空格，网页只会显示成一个，而小于号（<）和 大于号（>），网站则会认为是标签而无法直接显示在页面中。而这些都可以通过实体字符来解决；

同一个符号，可以用 “实体名称” 和 “实体编号” 两种方式引用。

显示 说明 实体名称 实体编号
< 小于 &lt; &#60;
> 大于 &gt; &#62;
& &符号 &amp; &#38;
" 双引号 &quot; &#34;
© 版权 &copy; &#169;
× 乘号 &times; &#215;
÷ 除号 &divide; &#247;
空格 &nbsp; &#160;
